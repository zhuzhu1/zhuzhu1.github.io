    String

    String类表示字符串。 Java程序中的所有字符串文字（例如“abc”）被实现为此类的实例。因为String对象是不可变的，所以它们可以被共享。String类被final修饰，不能被继承。

    StringBuilder

    一个可变的字符序列。 该类提供与StringBuffer兼容的API，但不保证同步。这个类被设计为在单线程使用字符串缓冲区的地方使用StringBuffer的代替替换（通常是这种情况）。 在可能的情况下，建议使用这个类优先于StringBuffer，因为在大多数实现中它将更快。

    每个字符串构建器都具有容量。 只要字符串构建器中包含的字符序列的长度不超过容量，则不需要分配新的内部缓冲区。如果内部缓冲区溢出，则会自动扩容。

    StringBuilder和StringBuffer都扩展了AbstractStringBuilder抽象类，AbstractStringBuilder类有两个变量，用来保存字符串中的字符。
