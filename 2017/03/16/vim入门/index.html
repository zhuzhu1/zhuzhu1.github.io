    ##插件
    参考《MacTalk人生元编程》是vim章节，下面是一些比较常用的插件。
    pathogen:管理插件的插件，pathogen在.vim目录下建立bundle文件，所有的插件都会在该目录下管理。
    NERDTree:可以在vim运行时显示目录和文件结构。
    command-T:是基于Ruby和C扩展实现的快速文件浏览的插件，类似Eclipse的Open Resource（command+shift+r)功能，可以通过模糊匹配快速定位并打开文件。
    Powerline:是vim的一个非常漂亮的状态栏插件。
